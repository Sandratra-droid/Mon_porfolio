À propos

Je suis Parson Sandratra, étudiant en M1 Informatique, avec une orientation vers l’administration des systèmes, des réseaux, du cloud et de la sécurité. Mon objectif est de travailler dans une entreprise où je pourrai participer à la mise en place, au suivi et à la sécurisation d’infrastructures informatiques.

J’aime travailler sur :

  • Installation et configuration de serveurs (Windows Server, Linux, environnements virtuels).
  • Topologies réseau (LAN, VLAN, routage, VPN, pare‑feu, sécurité réseau).
  • Environnements cloud (VPS, infrastructures AWS, hébergement web).
  • Surveillance et dépannage réseau (Zabbix, dépannage de coupures).
  • Sécurité offensive & défensive (test d’intrusion, SIEM, journaux, détection d’incidents).

Projets

Projet 1 : Infrastructure Active Directory + pfSense + WordPress e‑commerce

Mise en place d’une infrastructure complète incluant un domaine Active Directory (contrôleur de domaine, unités d’organisation, GPO), un pare‑feu pfSense et un site e‑commerce WordPress hébergé sur Windows Server 2019.

  • Technos : Windows Server 2019, Active Directory, GPO, UO, pfSense, firewall, routage, IIS/Apache, WordPress
  • Tâches : installation et configuration d’AD, gestion des utilisateurs et groupes, déploiement d’un serveur web, sécurisation via pfSense, hébergement d’un site e‑commerce
  • Résultat : domaine centralisé, environnement sécurisé, site web accessible et fonctionnel, intégration réseau/serveur/web

Projet 2 : Test d’intrusion et SIEM avec ELK (attaque & défense)

Projet de cybersécurité couplant des tests d’intrusion (Suricata, psexec, Mimikatz) et une plateforme SIEM basée sur la pile ELK (Elasticsearch, Logstash, Kibana).

  • Technos : Suricata, psexec, Mimikatz, ELK Stack, journalisation, analyse de logs
  • Tâches : scénarios d’attaque controlés, capture de logs, configuration de règles de détection, visualisation via Kibana pour analyser les comportements malveillants
  • Résultat : capacité à détecter des intrusions en temps quasi réel, rapport de sécurité et retour d’expérience sur les failles exploitées et corrigées

Projet 3 : VPN site‑to‑site pfSense + Tailscale et supervision Zabbix

Mise en place d’un VPN site‑to‑site avec pfSense complété par Tailscale, et mise en place d’un système de supervision réseau avec Zabbix pour surveiller les serveurs et postes clients.

  • Technos : pfSense, VPN site‑to‑site, Tailscale, Zabbix, agents Zabbix, monitoring
  • Tâches : configuration du VPN entre sites, sécurisation des flux, installation et configuration de Zabbix, déploiement d’agents, création de dashboards et alertes
  • Résultat : connexion sécurisée entre sites, visibilité complète sur l’état du réseau, détection rapide des pannes ou surcharges

Projet 4 : Téléphonie IP avec Asterisk et déploiement réseau terrain

Mise en place d’un système de téléphonie IP avec Asterisk sous Ubuntu Server et déploiement d’une infrastructure réseau (routeur Starlink, switches, access points) pour permettre aux clients d’accéder à Internet.

  • Technos : Asterisk, VoIP, SIP, Ubuntu Server, Starlink, switches, access points, déploiement terrain
  • Tâches : configuration du serveur Asterisk, gestion des comptes SIP, déploiement et configuration des équipements réseau, dépannage de coupures réseau en entreprise
  • Résultat : système téléphonique IP fonctionnel, réseau local stable, clients connectés à Internet, disponibilité améliorée des services réseau

Projet 5 : Hébergement web, VPS et infrastructure cloud AWS

Déploiement d’applications web avec hébergement sur LWS, configuration d’un VPS et mise en place d’une infrastructure cloud basique avec AWS.

  • Technos : LWS, VPS Linux, DNS, SSL, AWS (EC2, VPC, S3 ou équivalent), virtualisation, sécurité cloud
  • Tâches : installation et configuration du VPS, déploiement d’un site web, gestion des certificats SSL, mise en place d’une infrastructure cloud (réseaux, instances, stockage)
  • Résultat : environnement web hébergé, accessible et sécurisé, expérience pratique avec les services cloud et les bonnes pratiques de sécurité du cloud
